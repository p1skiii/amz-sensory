1) 首页重设计总原则（先把方向钉死）
WeChat 的定位（你要的高级感必须这样）

保留但弱化：只放在 Header 的一个小链接 + Footer 的一个小链接（甚至只写 “WeChat” 两个字）。

首页不出现“加入/联系我们/下单/扫码”这类词。

不做“CTA 卡片”（你现在那个大框“加入微信”就是典型电商/私域味道，必须删）。

首页的叙事结构（Tim 那种“主页即宣言”）

你要的是：一页读完就懂 AMZ 的价值系统

风土与关系（Where/Who） → 可持续（Why ethically） → 烘焙方法论（How） → 风味语言（What it tastes like） → 作品集（Library）

4) 首页文案骨架（繁中为主 + 英文同构）

下面每段都很短，符合你“信息密度高、排版强”的画册逻辑。

4.1 Manifesto（2 列：左文右图）

繁中：

标题：我們的咖啡不是更濃，而是更清。

三句短文（每句一行）：

風土是前提，關係是證據。

可持續不是口號，是一種節制。

烘焙是把結構留住，把多餘移走。

EN：

Title: Not stronger. Clearer.

Lines:

Terroir sets the stage. Relationships prove it.

Sustainability is restraint, not a slogan.

Roasting keeps structure — and removes excess.

4.2 Pillars（三栏）

Structure：酸不是尖，花不是飄。結構讓風味站得住。

Light：極淺不是薄，是把亮度留給香氣。

Clean：尾韻乾淨，是我們對每一次調整的底線。

（EN 同构，短句就好）

4.3 画像叙事（3 张 portrait + 3 段短文）

你说想“用画像 + 简单描述构造故事感”，这块就是核心。

每张图下面只允许 2–3 行：

誰（Producer / Roaster / Cupper）

他在意什麼

我們如何把它帶到杯中

示例（繁中）：

Producer：他在意成熟度與密度。 / 我們只買我們真正理解的批次。

Roasting：我們在意溫區，而不是火力。 / 層次靠節奏，不靠焦糖。

Cupping：我們用清晰度做裁判。 / 花果先到，但必須站得穩。

4.4 Terroir & Relationships（2 列大图 + 文本）

标题：Where / Who

三个 bullet（短、像杂志）：

來源可追溯（traceable）

關係長期（long-term）

描述可驗證（verifiable）

4.5 Sustainability（脚注式）

像 Tim 那种“信息可信但不营销”，用细线表格/条目：

Packaging：铝瓶/密封逻辑（你之前写过很强）

Buying：只写“我们如何做选择”，不写“我们很环保”

Waste：克制用词，避免口号

4.6 Roasting Method（结构化版式）

标题：How we roast
下面用 4 行“方法卡片”（不是正文长文）：

Temperature Bands：分層而不是堆料

Development：保留香氣的骨架

Solubility：讓萃取更可控

Finish：乾淨收尾

4.7 Current Releases（Library Preview）

2 列大图（你已经有 CoffeeCard 的风格雏形）
不要显示价格（你不是电商）
只显示：

名称

产地/处理

3 个 tasting words（hover 才出现）
